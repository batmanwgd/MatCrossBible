In order to translate the interface of the App into another language, make a copy of one of
the existing .tr files (for example English.tr), give it a proper name, open it in a file
editing program such as Notepad or Notepad++ and translate the text. Make sure you keep the
special symbols and tags and try to respect the length of the sentences.
The new translation will appear inside the App after the next restart and can be selected
from the settings.

The file must be in UTF-8 format!
